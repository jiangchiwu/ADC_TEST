# 6通道ADC采样、触发、FFT测试报告
## 测试概述
- **测试时间**：2026-05-25
- **测试对象**：STM32H750 6通道ADC采样系统
- **硬件配置**：PA5经运放分6路输入至6个ADC通道
- **通道映射**：
  - CH1: PC0
  - CH2: PC1
  - CH3: PC2
  - CH4: PC3
  - CH5: PA3
  - CH6: PA4

## 测试内容
1. 6通道ADC采样验证
2. 信号触发检测
3. FFT基波频率计算
4. 同通道去重功能（25ms）
5. 多频率信号测试
